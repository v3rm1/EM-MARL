## Overview

FireSim is an emergency response multi-agent reinforcement learning simulator built on [PettingZoo](https://pettingzoo.farama.org/). It provides a realistic simulation of disaster response scenarios where multiple types of agents must coordinate to handle various emergency situations.

### Key Features

- **Multi-Agent Coordination**: Support for collaborative and hierarchical agent structures
- **Diverse Agent Types**: Medics, Fire Force, Police, and Civilians with unique capabilities
- **Map-Based Simulation**: Continuous coordinate system with zones and dynamic incidents
- **Resource Management**: Limited resources that agents must manage strategically
- **Realistic Movement**: Physics-based movement with stamina, running, and terrain considerations
- **PettingZoo Compatible**: Full AEC environment API for RL training

### Use Cases

- Training multi-agent reinforcement learning algorithms
- Testing coordination strategies between heterogeneous agents
- Simulating emergency response scenarios
- Researching resource allocation in crisis situations
